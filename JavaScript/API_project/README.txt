Install:
	
	1.release the .zip file in a single folder and make sure
all the files have been placed in the correct directories.

	file list:	________________________________________________
			|
			|	./
			|		server.js
			|       	package.json
			|       	package-lock.json
			|		README.text
			|
			|		html/
			|			favicon.ico
			|			style.css
			|			food.js
			|________________________________________________

    2. run a shell in the root directory and install express module:

            npm install

====================================================================
Launch the server:
	
	starting a shell at the root directory and typing in the command:
	
		node server.js
	or
	    npm test

====================================================================
